# built-in-saudi.com

Static site for **built-in-saudi.com**, hosted on GitHub Pages.

## Hosting
- Served from the `main` branch root via GitHub Pages.
- Custom domain: `built-in-saudi.com` (apex) with `www` redirecting to apex.
- HTTPS enforced.

## Files
- `index.html` — landing page.
- `404.html` — not-found page.
- `CNAME` — custom domain for GitHub Pages.
- `robots.txt` / `sitemap.xml` — search engine directives.

## Analytics
Google Analytics 4 (`gtag.js`) is embedded in `index.html`.
